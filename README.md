# RandomProjects
I will create random project based on environment needs

## Projects

### Medium POC
A simple Proof of Concept (POC) for a Medium-like blogging platform.

#### How to Run
1. Ensure you have Java 17+ and Maven installed.
2. Navigate to the project directory: `cd medium-poc`
3. Run: `mvn spring-boot:run`
4. Open `http://localhost:8080` in your browser

For more details, see [medium-poc/README.md](medium-poc/README.md).

### Parking Meter POC
