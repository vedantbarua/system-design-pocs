# Improvements

- Add calendar sync for appointments, rides, home visits, and recurring care windows.
- Add SMS confirmations for caregivers who do not use the web dashboard.
- Add medication, vitals, and pharmacy integrations with explicit privacy controls.
- Add role-based household sharing for family, professional caregivers, and backup contacts.
- Add mobile offline mode for visits where connectivity is poor.
- Move care scans and reminder dispatch to dedicated background workers.
- Add retention policies for old care logs, handoffs, and audit history.
