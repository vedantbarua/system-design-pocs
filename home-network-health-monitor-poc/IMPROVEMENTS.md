# Improvements

## Production Gaps

Add authenticated agents, DNS/throughput tests, configurable thresholds, maintenance windows, ISP SLA reports, and notification preferences.

## Reliability Improvements

Add watermarks, allowed lateness, checkpoints, durable idempotency, transactional outbox writes, retry topics, and projection checksums.

## Scaling Improvements

Partition by household/target, use TimescaleDB hypertables, compute incremental rollups, split workers, and tier old telemetry.

## Security Improvements

Add OIDC, household isolation, device certificates, credential rotation, metadata encryption, and restricted replay controls.

## Testing Improvements

Add infrastructure integration tests, property tests for replay stability, event-order permutations, alert-storm tests, and Playwright responsive flows.
