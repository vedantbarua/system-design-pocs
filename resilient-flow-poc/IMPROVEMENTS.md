# Trading Log POC Improvements

## Product
- Add watchlists, portfolio grouping, and trader-level filtering.
- Support commissions, fees, and realized net P&L.
- Add order lifecycle states beyond executed trades.

- Persist trade and price events with replay support.
- Introduce event streaming for near-real-time dashboards.
- Add automated reconciliation jobs and import/export.

## UX
- Add sparkline charts for P&L and exposure.
- Add inline filters for symbol, trader, side, and time.
- Add CSV upload for bulk trade ingest.
