What This POC Demonstrates

Bit Masking

The generator packs timestamp, worker ID, and sequence into one positive 64-bit integer and exposes the exact binary slices used to decode it.

Time-Ordered IDs

Higher timestamp bits dominate the sort order, so IDs remain naturally sortable without a central sequence database.

Clock Drift Handling

Minor backward drift is absorbed by pinning generation to the last safe timestamp. Larger regressions are rejected so uniqueness is not silently compromised.

Worker State

Each worker tracks its last generated timestamp and sequence. Clock regressions are counted so you can see when a node had to rely on the drift guardrail.
